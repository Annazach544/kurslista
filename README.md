# Kurslista – Angular
Detta är en Angular-applikation som visar kurser från Webbutvecklingsprogrammet.

## Funktioner
- Hämtar data från JSON med HttpClient
- Visar kurser i en tabell
- Sortering på kurskod, kursnamn och progression
- Sökruta som filtrerar i realtid

## Tekniker
- Angular
- TypeScript
- HttpClient
- Services
- Signals
